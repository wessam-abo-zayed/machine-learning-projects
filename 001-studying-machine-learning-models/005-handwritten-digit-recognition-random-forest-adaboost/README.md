# Handwritten Digit Recognition using Random Forest and AdaBoost

This project demonstrates handwritten digit classification using machine learning techniques. The dataset is explored, preprocessed, and used to train and evaluate two supervised learning models:

- Random Forest Classifier
- AdaBoost Classifier

The goal is to compare the performance of both algorithms for recognizing handwritten digits.

## Dataset

The project uses a handwritten digit dataset containing numerical image representations of digits (0–9).

The dataset is used to:

- Explore data characteristics
- Visualize digit samples
- Train classification models
- Evaluate prediction performance

---

## Libraries Used

- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

---

## Workflow

### 1. Data Exploration

- Load dataset
- Check data dimensions
- Visualize sample digits
- Analyze class distribution

### 2. Data Preparation

- Split data into training and testing sets
- Prepare features and labels

### 3. Random Forest Model

- Train Random Forest classifier
- Predict digit classes
- Evaluate performance

### 4. AdaBoost Model

- Train AdaBoost classifier
- Predict digit classes
- Evaluate performance

### 5. Model Comparison

The performance of both models is compared using:

- Accuracy Score
- Confusion Matrix
- Classification Report

---

## Results

The notebook demonstrates how ensemble learning methods can achieve high accuracy for handwritten digit recognition.

Random Forest and AdaBoost are evaluated and compared to determine which model performs better on the dataset.

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/handwritten-digit-recognition-random-forest-adaboost.git
```

2. Navigate to the project directory:

```bash
cd handwritten-digit-recognition-random-forest-adaboost
```

3. Install dependencies:

```bash
pip install numpy pandas matplotlib seaborn scikit-learn
```

4. Launch Jupyter Notebook:

```bash
jupyter notebook
```

5. Open:

```text
handwritten-digit-recognition-random-forest-adaboost.ipynb
```
